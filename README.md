# ⚡ Token Speed Visualizer

A professional, real-time visualization tool built with **HTML**, **JavaScript**, and **Tailwind CSS** to compare how different Large Language Model (LLM) token generation speeds "feel" to an end-user.

## 🌟 Features

- **Side-by-Side Race**: Visualize 3 different generation speeds simultaneously to compare them.
- **Adjustable Parameters**: Customize the Tokens Per Second (TPS) for each panel.
- **Client-side Simulation**: Simulates the token streaming feel entirely in the browser, no server required.
- **Clean UI**: A modern dark-mode interface with real-time token counters and animations.

## 🚀 Getting Started

Since this is a static website, there is no installation required.

### How to Run
1. Clone or download the project files.
2. Open `index.html` in any modern web browser.

## 🎨 Technologies Used
- **HTML5 & JavaScript**: Core logic and structure.
- **Tailwind CSS**: Utility-first CSS for a modern UI.
- **Embedded Content**: Sample text is included directly in the application for a seamless "single-file" experience.
